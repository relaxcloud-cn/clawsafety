# ClawSafe 判定标准

## 三档结论

| 结论 | 含义 | 处置 |
|------|------|------|
| **恶意** | 存在明确恶意行为 | 立即删除，不要使用 |
| **有风险** | 存在无法被声明用途解释的可疑行为 | 人工审查后决定是否使用 |
| **安全** | 所有行为均可被声明用途合理解释 | 可以使用 |

## 恶意指标（命中任一即判定"恶意"）

- 情报命中：Intel Studio / ip-analysis / domain-analysis 确认为恶意
- C2 通信：代码中包含已知 C2 IP 或恶意域名
- 数据窃取：读取敏感文件并发送到外部
- 反弹 shell：nc -e、/dev/tcp/、mkfifo、socat exec 等
- 恶意发布者：author 在已知恶意发布者列表中
- 后门植入：条件触发的隐藏恶意代码
- 混淆执行：base64 编码 + eval/exec 组合

## 风险指标（命中任一即判定"有风险"）

- 意图-行为不匹配：行为无法被声明用途合理解释
- 权限过度：请求权限远超功能所需
- 不安全安装：curl | bash、未校验的外部下载
- 持久化机制：crontab、LaunchAgent、systemd，且与功能不匹配
- 未锁定依赖：>= 或无版本约束
- 社工话术：诱导用户执行命令
- Prompt 注入：操纵 agent 行为的指令

## 核心原则

**行为本身不决定风险，行为与用途的匹配度才决定风险。**

| 行为 | 安全扫描器 | 代码格式化工具 |
|------|-----------|--------------|
| 访问 ~/.ssh/ | 合理 | → 恶意 |
| 内嵌 C2 IP 列表 | 合理 | → 恶意 |
| 发送网络请求 | 视功能而定 | → 有风险 |
| LaunchAgent | 视功能而定 | → 有风险 |

## 情报优先

情报源（Intel Studio、ip-analysis、domain-analysis）给出明确结论时，直接采信，不二次判断。
