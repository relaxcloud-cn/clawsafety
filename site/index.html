What It Scans

Comprehensive security analysis

Injection (INJ)

Shell injection, SQL injection, reverse shells, command execution via unsanitized inputs.

4 rules

Secrets (SEC)

Hardcoded passwords, API keys, private keys, tokens left in skill configurations.

4 rules

Dependencies (DEP)

Unpinned versions, known CVEs, curl-pipe-bash patterns, suspicious package sources.

4 rules

Permissions (PRM)

Excessive file access, sensitive path reads, environment variable abuse, privilege escalation.

4 rules

Config (CFG)

Missing SKILL.md, prompt injection risks, insecure default configurations.

4 rules

Quick Start

Start scanning in seconds

# Install
$ cargo install clawsafety
# Scan a single skill
$ clawsafety scan ./my-skill/
# Scan all skills in a directory
$ clawsafety scan-all ./skills/
# Output as JSON for CI/CD
$ clawsafety scan ./my-skill/ --format json
